# Sarcasm-Detection-Training
Training a keras model to recognize sarcasm in order to classify an incoming text as sarcastic or the other.

## Requirements
- Python 3 or higher.

## Packages

### Numpy

py -m pip install numpy
```

### Tensorflow

py -m pip install tensorflow
```
